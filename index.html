Welcome to Preview it !!!

Congratulation, you are discovering a gorgoous and incredible chrome extension, which can make you capable of doing many things you have never done before.

Want to know more about it? put your mouse on me, and press 'w' key silently and with elegance.

How to setup

  1. Download it from the link on the right.
  2. Opne Chrome -> tools -> extensions
  3. Drag the extension file you just download into Chrome, it will setup automatically then.

How to use

1. Hover on the link you want to preview

Don't know what is hover? hover is putting your cursor onto the link(Don't need to click it!), that means the cursor in within the link range.

2. Press the "w" button on your keyboard.

Using the phone? Sorry, you are not able to use any kinds of extension.

3. interact with the preview box

Yes, you can simply scroll, click the links or even fill the forms in the preview box. You just need to regard it as a small window, just looks cute and not takes too much spaces.

4. When you don't want it anymore, just press "w" key again or check the close button.

The page on the top will disappear~
